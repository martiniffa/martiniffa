== Description ==

The plugin will add to your blog a table with exchange rates. The table can be placed as widget or shortcode [adce_currency_table]. Courses are updated by using server Tjenestetorget.

Important! On the weekends Forex market does not work so dynamically, so exchange rates may not change value.
Courses provides Tjenestetorget

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place a widget or shortcode [adce_currency_table] on your blog
